# SkillUp - Employee Training Platform

A premium React-based training platform for upskilling employees.

## Features
- **Course Catalog**: Filter by category (Frontend, Backend, Mobile, Sales, HR).
- **Interactive Quizzes**: Test your knowledge with immediate feedback.
- **Assessments**: Comprehensive skill checks with progress tracking.
- **Certificate Generation**: Earn certificates upon completion.
- **Premium UI**: Modern glassmorphism design with smooth animations.

## Tech Stack
- React 18
- Vite
- Framer Motion (Animations)
- React Router DOM
- LUCIDE React (Icons)
- Vanilla CSS (Custom Design System)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at `http://localhost:5173`

## Project Structure
- `src/components`: Reusable UI components (Navbar, etc.)
- `src/pages`: Main application views (Home, CourseList, Player, Quiz).
- `src/data`: Mock data for courses and questions.
- `src/index.css`: Global design system and variables.
